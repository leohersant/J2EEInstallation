How to install JDK and Eclipse for J2EE version with tomcat 9 on ubuntu 20.04

# Install JDK (java)

  • download debian package: https://www.oracle.com/java/technologies/javase/jdk14-archive-downloads.html
  • Install it. It contains JRE already

# Install Eclipse IDE

  • sudo apt install --classic eclipse

Update classic version for it to be same as Eclipse for J2EE version.

1.In Eclipse go Help -> Install new Software...
2.Press Add...
3.In address enter http://download.eclipse.org/releases/indigo/ and name - whatever you like.
4.Install JavaEE Developer Tools (under "Web, XML, JavaEE, and OSGi Enterprise Tools")
5(Optional, but very useful).Install Marketplace Client (General Purpose Tools -> Marketplace Client).

# Install tomcat

  • sudo apt install tomcat9 tomcat9-admin

# fix directories for Eclipse IDE

  • cd /usr/share/tomcat9
  • sudo ln -s /var/lib/tomcat9/conf conf
  • sudo ln -s /var/log/tomcat9 log
  • sudo ln -s /etc/tomcat9/policy.d/03catalina.policy conf/catalina.policy
  • sudo chmod -R a+rwx /usr/share/tomcat9/conf

# allow startup from inside eclipse

  • sudo service tomcat9 stop
